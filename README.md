1.16.1 Boat Clutch practice map - Short demonstration video: https://youtu.be/_u_oVvMdfzE

Simply run off the spawn platform and boat clutch. For higher or lower boat clutch heigts use the command blocks.

To respawn switch to your 6 slot.

If you place the boat but die you can press the middle button to remove all placed boats and receive a new one.

When quickly respawning and jumping off again be aware of minecrafts 3 second immunity window after respawning. So for lower heights wait a short moment before jumping again.

Feel free to go in creative mode and adjust the hotbar to your liking. 

If you want another specific height you can change the values in the command blocks or just let me know and I will add them to the next version of the map.

If you want any new features or changes please comment them under the map demonstration video.
